Principais Tags HTML

Menu de navegação (exemplo)

<head>: Contém metadados da página, como título, meta-descrição, e links para folhas de estilo.

<body>: Contém o conteúdo visível da página, incluindo texto, imagens, vídeos e outros elementos.

<h1> - <h6>: Define títulos de níveis diferentes.

<p>: Define um parágrafo de texto.

<img>: Insere uma imagem na página.

<a>: Cria um hiperlink para navegar para outra página ou parte da mesma página.

<form>: Cria um formulário para interação do usuário.

<table>: Cria uma tabela para organizar dados em linhas e colunas.

<nav>: Define uma seção de navegação da página.

<header>: Define o cabeçalho da página ou de uma seção.

<footer>: Define o rodapé da página ou de uma seção.

Exemplo de Uso do Texto Alternativo (alt)

Abaixo temos duas imagens: uma sem descrição adequada e outra com um texto alternativo descritivo.

Observações

  • O atributo alt é essencial para acessibilidade, pois descreve o conteúdo da imagem para leitores de tela.
  • Se a imagem for decorativa, pode-se usar alt="" para que ela seja ignorada por tecnologias assistivas.
  • Se a imagem tiver função informativa, o alt deve descrever de forma clara o que ela representa.
